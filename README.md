# Fast Clip
<img width="100" height="100" alt="icon2" src="https://github.com/user-attachments/assets/2bdcbc83-10b2-43d4-bf3b-548ccfedb555" />

A simple video editor, helps you create clips fastly.

## Features
- **FAST**: We use real time design, which can open big video files without prescanning.
- **PERFORMANCE**: Powered by rust and gpui, bring fast software reaction.
- **FORMAT**: Based on ffmpeg, bring good compatibility with video formats.
